⚙️ Konfiguracja
📋 Jak uzyskać klucz Google Vision API
  1. Wejdź na console.cloud.google.com
  2. Zaloguj się kontem Google
  3. Kliknij „Select a project" → „New Project" (np. nagrobki-ocr)
  4. APIs & Services → Library → szukaj „Cloud Vision API" → Enable
  5. APIs & Services → Credentials → „+ Create Credentials" → „API key"
  6. Skopiuj klucz (AIzaSy...)

💰 Pierwsze 1000 zapytań/miesiąc za darmo + $300 kredytów na start

Jeśli podasz URL Workera — dane pójdą przez niego (z haszowaniem IP).
Jeśli pusty — wysyłka bezpośrednio do Google Sheets.
