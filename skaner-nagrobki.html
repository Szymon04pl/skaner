⚙️ Konfiguracja
📋 Jak uzyskać klucz Gemini API (za darmo) — SZCZEGÓŁOWA INSTRUKCJA

⚠️ WAŻNE: Ta aplikacja używa modelu gemini-1.5-flash-latest. Klucz musi mieć dostęp do tego modelu.

KROK 1: Utwórz projekt Google Cloud

  1. Wejdź na console.cloud.google.com
  2. Zaloguj się kontem Google
  3. U góry strony kliknij na nazwę projektu (obok loga Google Cloud)
  4. Kliknij „Nowy projekt" (lub „New Project")
  5. Nazwij go np. skaner-nagrobkow i kliknij „Utwórz"
  6. Poczekaj ~15 sekund aż projekt się utworzy, potem wybierz go u góry

KROK 2: Włącz Generative Language API

  1. Wejdź na tę stronę (Generative Language API)
  2. Upewnij się, że u góry widnieje Twój nowy projekt
  3. Kliknij niebieski przycisk „Włącz" / „Enable"
  4. Poczekaj aż API się aktywuje (~30 sekund)

KROK 3: Utwórz klucz API w Google AI Studio

  1. Wejdź na aistudio.google.com/apikey
  2. Kliknij „Create API key"
  3. Z listy wybierz projekt, który przed chwilą utworzyłeś (np. „skaner-nagrobkow")
  4. Kliknij „Create API key in existing project"
  5. Skopiuj klucz — zaczyna się od AIzaSy...

KROK 4: Sprawdź czy klucz działa

  1. Wklej klucz w pole poniżej i kliknij 💾
  2. Kliknij przycisk „🧪 Testuj klucz" który pojawi się obok
  3. Jeśli widzisz ✅ — klucz działa, możesz skanować
  4. Jeśli widzisz ❌ — sprawdź czy:
    • Klucz jest poprawnie skopiowany (bez spacji)
    • Generative Language API jest włączone (Krok 2)
    • Wybrałeś właściwy projekt przy tworzeniu klucza
    • Poczekaj 2-3 minuty — nowe klucze potrzebują czasu na aktywację

💰 Darmowe limity Gemini 1.5 Flash:

15 zapytań/minutę • 1500 zapytań/dzień • 1 milion tokenów/minutę

❌ Najczęstsze problemy:

  • 429 Too Many Requests — poczekaj 1-2 minuty i spróbuj ponownie (limit się resetuje)
  • 403 Forbidden — Generative Language API nie jest włączone (Krok 2)
  • 400 Bad Request — klucz jest uszkodzony lub zdjęcie za duże (maks ~20MB)
  • 404 Not Found — klucz nie ma dostępu do modelu (powtórz Krok 2)

Jeśli podasz URL Workera — dane pójdą przez niego (z haszowaniem IP).
Jeśli pusty — wysyłka bezpośrednio do Google Sheets.
